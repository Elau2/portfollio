Image Manipulation Project

March 2018, Project Lead the Way Computer Science and Software Engineering

Partner(s): Subash Shibu

The Image Manipulation Project was designed to help families make creative and memorable edits to exsisting pictures. Some examples of image manipulation functions include colored tint, a realistic pencil sketch, etc. I learned basic python code and how to manipulate images using such code. Both members of the project could have allocated better time management, but both members completed fair amounts of work. I played an integral role in the tint functions, pencil sketch function, and ASCII art.
