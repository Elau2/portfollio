Grumpy Grandma

September 2017, Project Lead the Way Computer Science and Software Engineering

Partner(s): Safwan Kasim

Grumpy Grandma is one of the first game "programs" that I have made. I learned basic logic and thinking skills about writing code and basic documentation skills. The game was made using MIT Scratch, a drag and drop based coding system. The objective was to create a game based on a controllable "robber" that aims to avoid getting hit by a purse thrown by the "grandma." Each progressive level increases the speed of the purse, making it more difficult to dodge the purse. Each round lasts for 60 seconds. I had an integral role in creating the level and home screen parts of the game.
