Paint Pot

October 2017, Project Lead the Way Computer Science and Software Engineering

Partner(s): Soleimon Qazi

Paintpot is an app created using MIT App Inventor based on existing, standard painting apps. I further learned more logic and thinking skills, when it comes to programming, and basic documentation skills during the time I was developing this app. The aim of the app was to create an app that allows the editing of taken pictures, creating a function that allows the user to color any part of the picture using any color. I played an integral role in color choosing and basic coloring functions of the app.
